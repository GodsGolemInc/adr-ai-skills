# /pr Command

Create Pull Request with ADR integration.

## Usage
- `/pr` - Create PR (interactive)
- `/pr --draft` - Create draft PR
- `/pr --title "Title"` - With specified title
- `/pr status` - Check existing PR
- `/pr update` - Update PR description
