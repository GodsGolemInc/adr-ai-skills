# /jj-analyze Command

Analyze JJ changes to detect architectural decisions.

## Usage
- `/jj-analyze [change-id]` - Analyze specific change
- `/jj-analyze recent [n]` - Analyze recent N changes
- `/jj-analyze pending` - Show unprocessed candidates
