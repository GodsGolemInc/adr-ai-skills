# /constraints-check Command

Validate code against architectural constraints.

## Usage
- `/constraints-check [path]` - Check path or entire codebase
- `/constraints-check --adr ADR-XXXX` - Check specific ADR
- `/constraints-check --ci` - CI-friendly output
- `/constraints-check --update` - Update constraints from ADRs
